0) 100% branch code coverage
npm test


1) Once off execution, using 'npx':
npx @lit/artifact-generator


2) Install globally:
npm install -g @lit/artifact-generator


3) Now run from anywhere:
lit-artifact-generator


4) Provide input, and it generates an NPM module...
lit-artifact-generator -i https://schema.org/version/latest/schema.ttl
  ...in the './generated' folder. 
But that has ~2,000 terms! Maybe we only want 5.


5) Turn off the interactive prompting:
lit-artifact-generator -i https://schema.org/version/latest/schema.ttl --noprompt

 
6) Create a selective RDF file...
lit-artifact-generator -i https://schema.org/version/latest/schema.ttl --vocabTermsFrom select-from-schema.rdf --noprompt


7) Enhance our selected terms with multi-lingual translations.
lit-artifact-generator -i https://schema.org/version/latest/schema.ttl --vocabTermsFrom schema-inrupt-ext --noprompt


8) We can also automatically 'npm install' our modules...


9) ...or 'npm publish' them (to any registry), or just bump the version number of an already published version...
lit-artifact-generator -i https://schema.org/version/latest/schema.ttl --vocabTermsFrom schema-inrupt-ext --noprompt --bumpVersion major


10) Bundle multiple vocabs into one single module, using a YAML file to stipulate them all.
